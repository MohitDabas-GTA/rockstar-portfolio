ABOUT ME

WHO AM I?

My name is Mohit Dabas, and I am an aspiring software developer with a strong interest in web development, Python programming, and modern software technologies. I enjoy creating responsive websites, desktop applications, and automation tools while continuously expanding my programming knowledge through practical projects and self-learning.

My programming journey began with HTML and CSS, where I learned how websites are structured and designed. As my understanding grew, I started exploring JavaScript to build interactive web experiences and Python to create desktop applications, automation scripts, and utility programs. Every project I complete teaches me something new about software development, problem-solving, and writing cleaner code.

One of my biggest inspirations is the gaming industry. I admire how technology, creativity, and teamwork come together to build immersive digital experiences. This motivation encourages me to continue learning programming, improving my technical skills, and challenging myself with increasingly complex software projects.

I believe that programming is a continuous learning journey. Instead of only studying theory, I prefer building real projects that allow me to practice what I learn. Whether I am designing a website, creating a Python application, or experimenting with new technologies, every project helps me become a more confident and capable developer.

Through this portfolio, I want to showcase my work, document my learning journey, and share the projects that have helped me grow as a developer. I enjoy exploring modern development tools, improving user experiences, and creating software that is both functional and visually appealing.

I am currently focused on improving my knowledge of JavaScript, Python, Git, responsive web development, Flutter, and software engineering principles. My goal is to continue building meaningful projects, contribute to open-source communities, and prepare myself for a professional career in software development.

Thank you for taking the time to visit my portfolio. I hope this website gives you a clear understanding of my technical skills, my passion for programming, and my commitment to continuous learning. I look forward to creating more innovative projects and expanding this portfolio as I gain new experience.
